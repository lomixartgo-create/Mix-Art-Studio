Starter website for **Mix Art Studio**, a small logo and brand-kit business idea.

- Landing page
- Package/pricing section

- Simple process section
- Email call-to-action using `lomixartgo@gmail.com`

## Local preview

From this folder:

```bash
python3 -m http.server 8088
```

Then open:

```text
http://localhost:8088
```

On Bean's LAN, if the server is running and firewall allows it:

```text
http://192.168.2.250:8088
```
